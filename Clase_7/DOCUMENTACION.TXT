**Sistema de Control de Acceso a un Club Nocturno**

Crea un programa en PHP que determine si una persona puede entrar a un club nocturno basándose en los siguientes criterios:

1. Debes pedir al usuario:
   - Edad
   - Hora actual (en formato 24 horas)
   - Si tiene membresía VIP (verdadero/falso)
   - Cantidad de bebidas alcohólicas consumidas

2. Las reglas son:
   - Solo pueden entrar personas mayores de 18 años
   - El club abre de 21:00 a 03:00 horas
   - Personas VIP pueden entrar desde las 20:00
   - Si han consumido más de 3 bebidas alcohólicas, no pueden reingresar

3. Debes usar:
   - Variables para almacenar los datos del usuario
   - Constantes para la edad mínima y horarios
   - Diferentes tipos de datos (números, booleanos)
   - Operadores de comparación para verificar edad y hora
   - Operadores lógicos para combinar condiciones
   - Estructuras if/else para tomar decisiones
   - Un operador ternario para mostrar un mensaje final
